### 公共信息

<h3>token说明</h3>
<ul>
<li>
<p>注册</p>
</li>
<li>
<p>登陆</p>
</li>
<li>
<p>后端返回token</p>
</li>
<li>
<p>用户使用token请求</p>
<ul>
<li>这个token未过期且不是处于快过期的期限内——&gt;正常访问</li>
<li>这个token过期——&gt;后端返回401</li>
<li>这个token快要过期——&gt;后端将在响应头中添加Authorization,值是新的token,前端在获取后替换掉本地的token</li>
</ul>
</li>
<li>
<p>登陆后所有请求的Header中都要携带token,如果携带token将会在原来的放回信息中附加用户信息(画作是否收藏、画师是否关注、评论是否点赞)</p>
</li>
</ul>
