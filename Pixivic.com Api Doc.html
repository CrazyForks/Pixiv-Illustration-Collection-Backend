用户登录

基本信息

Path： /users/token

Method： POST

接口描述：

响应头会返回Authorization

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/json 是

Query

参数名称 是否必须 示例 备注
vid 是 验证码id
value 是 验证码

Body

名称 类型 是否必须 默认值 备注 其他信息
username string 非必须 用户名或者邮箱
password string 非必须 密码

返回数据

名称 类型 是否必须 默认值 备注 其他信息
message string 非必须
data object 非必须 用户信息
├─ id number 非必须 用户id
├─ username string 非必须 用户名
├─ email string 非必须 邮箱地址
├─ avatar string 非必须 头像url
├─ gender null 非必须 性别
├─ signature null 非必须 个性签名
├─ location null 非必须 地理位置
├─ star number 非必须 积分
├─ isCheckEmail boolean 非必须 是否校验邮箱
├─ isBindQQ boolean 非必须 是否绑定qq
